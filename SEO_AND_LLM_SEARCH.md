# SEO and LLM Search Notes

The current site structure follows a revenue-first SaaS SEO approach inspired by public Breaking B2B / Sam Dunning guidance:

- Prioritise bottom-funnel pages that match buyer intent.
- Build pages around specific use cases, verticals, comparisons and alternatives.
- Make commercial pages deep enough to answer buyer questions.
- Add clear FAQs and structured data so Google and AI answer engines can understand the page.
- Treat SEO and AI-search visibility as one system.
- Build external proof over time: software directories, industry listicles, partner mentions, client stories and relevant backlinks.
